# MaDarkDown (MaDark)
###### Version 0.0.1

MaDarkDown is a renderer, with the objective of a more recent view of the old Markdown. It proposes all the Markdown basic features (not counting the GitHub, Obsidian or even RMarkdown enhancements to the language) plus some new ones.
MaDarkDown is targeted to be a superset of HTML and not Markdown, making it as the same layer
